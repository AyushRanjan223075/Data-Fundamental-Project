# 📊 Student Performance Analysis
KIIT University | Data Fundamentals Project  
Author: Ayush Ranjan

---

## 📌 Project Overview
This project analyzes student academic performance using SQL, Excel, and Python. The goal is to identify top-performing students, analyze department-wise performance, detect at-risk students, and study the relationship between attendance and marks.

---

## 🛠️ Technologies Used
- SQL (SQLite)
- Microsoft Excel
- Python (Pandas, Matplotlib, Seaborn)

---

## 📂 Dataset
- student_dataset_1000.csv
- Contains:
  - Student ID, Name, Department
  - Marks (Math, Science, Programming)
  - Attendance (%)

---

## 🔍 Key Analysis
- Calculated average marks of students
- Identified top-performing student
- Department-wise performance comparison
- Detection of students scoring below 60
- Correlation between attendance and marks

---

## 📊 Visualizations
- Bar Chart: Student average marks
- Column Chart: Department performance
- Scatter Plot: Attendance vs Marks
- Heatmap: Correlation matrix

---

## 📈 Key Insights
- Top performer scored 100 in all subjects
- Civil department had highest average marks
- Several students identified as at-risk
- Attendance has weak correlation with marks
- Overall class average ≈ 71.37

---

## 🎯 Conclusion
The project shows that student performance depends more on understanding and study patterns rather than attendance alone. It helps identify students needing support and improves academic strategies.
